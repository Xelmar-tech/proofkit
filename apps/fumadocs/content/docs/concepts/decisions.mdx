## Decision packs (bundled choices)

Some decisions were made as bundles — changing one would ripple through the others:

### Toolchain pack (M1)

- **Turborepo** over nx/npm workspaces — lighter config, native to the pnpm ecosystem.
- **pnpm** over npm/yarn — strict dependency resolution, workspace protocol (`workspace:*`).
- **oxlint + oxfmt** over ESLint + Prettier — 50x/10x faster, single Rust toolchain.
- **Fumadocs** over Docusaurus/Nextra — Tanstack Start native, colocated in monorepo.
- **Changesets** over semantic-release — manual version bump prompts, changelog generation.

### Quality pack (M2)

- **Lefthook** over husky+lint-staged — single binary, native staged-file variable, `stage_fixed`.
- **Pre-commit**: staged-file-scoped, parallel format+lint, auto-format + auto-restage.
- **Pre-push**: whole-workspace, sequential check-types+check, hard-reject, no `--no-verify` carve-out.

### Plugin pack (M4)

- **Single verb** (`scaffold-proof`) for v0 — scope cap per EPIC.md.
- **Shared package** (`packages/plugin-shared/`) — Claude and Codex plugins consume same logic.
- **Single catalog** (`.claude-plugin/marketplace.json`) — Codex reads Claude's catalog format.
- **Vertical-plugin shape** — skills + commands, not agent personas.

See the individual ADRs for the full rationale behind each decision.
